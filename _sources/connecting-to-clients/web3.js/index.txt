**web3.js**

这是实现了 `Generic JSON RPC <https://github.com/ethereum/wiki/wiki/JSON-RPC>`_ 的以太坊兼容 `JavaScript API <https://github.com/ethereum/wiki/wiki/JavaScript-API>`_ 。它可以作为npm上的node模块，作为bower的可嵌入js模块以及meteor.js的一个包。

相关资源：

* GitHub: https://github.com/ethereum/web3.js/
* Wiki: https://github.com/ethereum/wiki/wiki/JavaScript-API
* Gitter: https://gitter.im/ethereum/web3.js
* NPM module: https://www.npmjs.com/package/web3
